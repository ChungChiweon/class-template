실기기 테스트 순서

        
    
          
  1. Supabase 프로젝트에서 `supabase/schema.sql`을 실행합니다.
    2. 
          
  2. `student-reports` Storage bucket과 Realtime 설정을 확인합니다.
    3. 
          
  3. `lib/supabase-config.js`에 URL과 anonKey를 입력하고 `enabled: true`로 바꿉니다.
    4. 
          
  4. 재배포 후 이 진단 화면에서 연결 테스트를 순서대로 실행합니다.
    5. 
          
  5. 태블릿에서 `/sessions/ai-practice/`, 강사 PC에서 `/present/`를 열어 학생 상태가 보이는지 확인합니다.
