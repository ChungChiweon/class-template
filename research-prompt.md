# GPT 심층연구 프롬프트

너는 “AI 활용 실습형 수업 웹페이지”를 설계하는 교육과정 기획자이자 수업 운영 컨설턴트다.

내 목표는 특정 수업 주제 하나에만 묶인 웹페이지가 아니라, 여러 수업 주제에 반복 적용할 수 있는 범용 수업 운영 템플릿을 만드는 것이다.

현재 사례는 “AI 호텔 홍보영상 제작 수업”이지만, 이후에는 다른 주제의 수업에도 같은 구조를 적용하려고 한다.

다음 관점에서 심층적으로 분석하고 설계안을 제안하라.

## 1. 범용 수업 템플릿 구조

다양한 실습형 수업에 공통으로 적용 가능한 웹페이지 구조를 제안하라.

반드시 포함할 영역:
- 수업 개요
- 전체 제작 흐름
- 교시별 수업 프로세스
- 단계별 실습 카드
- 도구 바로가기
- 프롬프트/자료 박스
- 예시 결과물
- 최종 제출물
- 리뷰/피드백
- 강사용 운영 메모

각 영역은 “학생용 정보”와 “강사용 정보”를 구분해서 설계하라.

## 2. 교시별 운영 구조

4교시 또는 6교시 수업에 모두 적용 가능한 구조를 제안하라.

각 교시는 다음 항목으로 정리하라.
- 교시 목표
- 학생이 해야 할 일
- 강사가 설명해야 할 일
- 사용할 도구
- 산출물
- 체크포인트
- 실패 시 대체 방법

## 3. 주제 교체가 쉬운 데이터 구조

웹페이지를 매번 새로 만들지 않고, 수업 주제별 데이터만 바꿔서 적용할 수 있는 구조를 제안하라.

예:
- 수업 제목
- 대상
- 수업 시간
- 도구 목록
- 단계 목록
- 프롬프트 목록
- 예시 이미지/영상
- 최종 결과물
- 리뷰 항목

가능하면 JSON 형태의 데이터 구조 예시를 제안하라.

## 4. AI 실습 수업에서 반드시 필요한 기능

실제 수업 시간에 사용할 웹페이지라면 어떤 기능이 필요한지 제안하라.

예:
- 아코디언 단계 진행
- 복사 가능한 프롬프트
- 이미지/영상 팝업
- 결과물 업로드 또는 미리보기
- 강사용 주의사항
- 학생 실수 방지 문구
- 도구별 바로가기
- 체크리스트
- 제출물 기준

각 기능이 왜 필요한지도 설명하라.

## 5. 학생 통제와 수업 운영 리스크

AI 도구 실습 수업에서 학생들이 자주 실수하는 지점을 정리하라.

특히 다음 항목을 포함하라.
- Generate 버튼을 너무 빨리 누르는 문제
- 모델 선택 실수
- 무료 크레딧 낭비
- 프롬프트 복사 누락
- 참조 이미지 업로드 누락
- 결과물 저장 실패
- 도구 UI가 계정마다 다른 문제
- 설치형 도구 다운로드 지연

각 문제에 대한 예방책과 웹페이지에 넣을 안내 문구를 제안하라.

## 6. 수업 주제별 변환 예시

아래와 같은 다른 수업 주제로 바꿀 때 어떤 항목이 바뀌고 어떤 항목은 유지되는지 예시를 들어라.

예시 주제:
- AI 호텔 홍보영상 제작
- AI 관광지 소개영상 제작
- AI 제품 광고영상 제작
- AI 진로 발표영상 제작
- AI 학교 홍보영상 제작
- AI 지역 축제 홍보영상 제작

각 주제별로 필요한 도구, 산출물, 프롬프트 유형, 평가 기준을 비교하라.

아래 형태로 결과를 제공하라.

4. 교시별 수업 운영안
5. 학생용 화면 구성
6. 강사용 화면 구성
7. 주제 교체 시 수정해야 할 항목
8. AI 실습 수업 운영 체크리스트
9. 향후 확장 기능 제안

답변은 실무형으로 작성하라.
